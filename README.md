## Kriteria:
- [x] Mampu Menambahkan Data Buku
- [x] Memiliki Dua Rak Buku
- [x] Dapat Menghapus Data Buku
- [x] Manfaatkan localStorage dalam Menyimpan Data Buku

## Fitur:
- Offline (Progressive Web App)
- DarkMode / LightMode
- Responsive 
- Cari Buku
- Edit Buku
- Hapus Buku
- Tambah Buku
